# to-do

A simple to-do list based on Rocketseat Ignite course and built with React, Typescript and Tailwind.

![image](https://user-images.githubusercontent.com/42921279/179120796-14bc2353-394b-463c-8163-531c31f3a7f3.png)

View demo: https://to-do-gamma-seven.vercel.app/

## Build and run

### To Run Locally:

Clone the repository and perform the following command line actions:

```bash
$> cd to-do
$> npm install
$> npm run dev
```
